# mosquitto

Lightweight MQTT broker for IoT device messaging.

- **Website:** https://mosquitto.org
- **Docs:** https://mosquitto.org/documentation
- **Access:** None (internal service consumed by Home Assistant and Zigbee2MQTT)

## Networks

| Network | Purpose |
|---|---|
| `home_automation` | Used by Home Assistant and Zigbee2MQTT to publish/subscribe |
| `uptime_kuma` | Health monitoring |
